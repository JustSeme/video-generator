- У нас есть список горячих тем для роликов с описанием
- Для генерации видео берём одну тему из списка

# Флоу генерации
1. Генерация сценария на основе темы:
- Массив объектов { id: uuid, title: string, text: string, visual: string, duration: number }


2. Генерация озвучки:
- Скармливаем text в ElevenLabs/Coqui TTS/Murf
- Полученный аудиофайл сохраняем c названием {id}.{ext}

3. Генерация визуала:
- Скармливаем visual в MidJourney/DALL·E 3/NanoBanano
- Полученные картинки сохраняем с названием {id}.{ext}

4. Композиция:
- С помощью FFMPEG совмещаем аудио и визуал по duration
- Добавляем субтитры (?)
- Добавляем анимации (?)
- Сохраняем видеофайл с названием {videoTheme}.mp4

5. Превью и миниатюра
- Генерируем превью на основе темы ролика в MidJourney/DALL·E 3/NanoBanano, сохраняем как {videoTheme}-preview.jpg
- Генерируем миниатюру на основе темы ролика в MidJourney/DALL·E 3/NanoBanano, сохраняем как {videoTheme}-thumbnail.jpg

6. Удаляем промежуточные файлы - озвучка, визуал

7. Руками загружаем видео на ютуб:
- Видео
- Превью
- Миниатюра
- Тема видео
- Описание видео
- Теги видео
